# Screenshots

Place your screenshots here before submission.

| File | Description |
|------|-------------|
| `desktop.png` | Full desktop view of the MedProof dashboard |
| `mobile.png` | Mobile-responsive view of the MedProof UI |

## How to add screenshots

1. Take a screenshot of the app running locally or on Vercel
2. Save as `desktop.png` (recommended: 1280×800px)
3. Save as `mobile.png` (recommended: 390×844px — iPhone 14 size)
4. Commit and push — they will appear in the README automatically
